# Proyecto Olist: Análisis del Mercado de E-Commerce en Brasil

¡Bienvenido al proyecto Olist! 

En este proyecto, analizaremos datos del mercado para ayudar a la empresa Olist a evaluar la posibilidad de expandirse a este país. A través de la exploración y análisis de los conjuntos de datos proporcionados, generaremos un informe que permita proyectar el comportamiento del mercado de E-Commerce en Brasil. A continuación, se detallan los objetivos específicos, el stack tecnológico utilizado y las necesidades del proyecto.





## Objetivos

- Análisis de la evolución de ventas según la segmentación del cliente: 
- Exploraremos las tendencias de compra de los clientes año tras año, mes a mes, para comprender la evolución de las ventas. 
- Este análisis proporcionará información importante para el informe del cliente.

- Análisis del crecimiento de las categorías de productos: 
- Mediante el análisis de las categorías de productos y su relación con las ventas, identificaremos las categorías con mejor desempeño y aquellas que están experimentando un crecimiento o disminución. 
- También consideraremos la influencia de los costos de envío en las ventas.

- Análisis de logística y métricas de entrega: 
- Para ingresar al mercado brasileño, el tema de la logística es crucial. 
- Realizaremos un análisis de las principales métricas de logística para comprender los desafíos y oportunidades en este aspecto.


## Stack Tecnológico

Durante el desarrollo de este proyecto, utilizaremos las siguientes tecnologías y herramientas:

- Power BI: Utilizaremos Power BI para el diseño y creación de un tablero de visualización de datos interactivo y dinámico.
- MySQL: Utilizaremos MySQL como base de datos para almacenar y gestionar los datos del proyecto.
- Google Colab: Utilizaremos Google Colab como entorno de desarrollo para el análisis exploratorio de datos, modelado predictivo y visualización de datos.
- Matplotlib, Seaborn: Utilizaremos estas bibliotecas de visualización de datos en Python para crear gráficos y visualizaciones informativas.
- Numpy, Pandas: Utilizaremos estas bibliotecas de Python para la manipulación y análisis de los datos.
- Scikit-learn: Utilizaremos Scikit-learn para el modelado predictivo y construcción de modelos de machine learning, si es necesario.
- GCP (Google Cloud Platform): Podemos utilizar GCP para el almacenamiento y procesamiento de grandes volúmenes de datos, si es necesario.

## Necesidades

El proyecto requerirá abordar las siguientes necesidades:
- Limpieza y preparación de datos: Realizaremos tareas de limpieza y transformación de datos para asegurar su calidad y coherencia.
- Análisis exploratorio de datos: Exploraremos los conjuntos de datos proporcionados, identificando patrones, tendencias y relaciones relevantes entre las variables.
- Modelado predictivo: Si es necesario, emplearemos técnicas de modelado predictivo para realizar predicciones sobre el comportamiento futuro del mercado de E-Commerce en Brasil.
- Visualización de datos: Utilizaremos gráficos y visualizaciones claras y efectivas para comunicar los hallazgos y resultados del análisis de manera comprensible.

## Alcance del Proyecto

El alcance del proyecto incluirá:
- Análisis de datos de ventas y segmentación del cliente.
- Desarrollo de indicadores de rendimiento clave (KPIs) para evaluar el desempeño del mercado.
- Generación de un informe con conclusiones y recomendaciones basadas en el análisis de los datos.
- Diseño y creación de un tablero interactivo en Power BI para visualizar los resultados y facilitar el acceso a la información relevante.

## Limitaciones del Proyecto

Es importante tener en cuenta las siguientes limitaciones del proyecto:
- Tiempo: El proyecto se realizará dentro de un marco de tiempo específico y se enfocará en los datos pre pandémicos. Esto significa que no se tendrán en cuenta los efectos de la pandemia en el mercado de E-Commerce en Brasil.
- Datos pre pandémicos: Los conjuntos de datos proporcionados abarcan el período desde 2016 hasta 2018, lo que significa que los datos son anteriores a la pandemia de COVID-19. Por lo tanto, no se considerarán los posibles cambios o impactos en el comportamiento del mercado debido a la pandemia.
